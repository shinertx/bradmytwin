# PRD

## Product
Brad.com Omnichannel Digital Twin MVP.

## Objective
Provide one persistent AI twin per user identity across SMS, WhatsApp, Telegram, and Web while enforcing strict user isolation and approval-gated write actions.

## Success Criteria
1. New user enters from channel and is identity-resolved.
2. User can complete onboarding with phone anchor.
3. User can connect Google account.
4. User can request scheduling action.
5. Event creation only proceeds after approval.
6. Multiple users can run concurrently without leakage.

## Non-goals
- Payments
- Voice calls
- Autonomous purchases
- Native mobile app
- Advanced long-term memory
